
# Web Technologies Assignments

This repository contains my solutions to the assignments from the Web Technologies course. The exercises cover topics such as:

- HTML
- CSS
- JavaScript
- Responsive design
- Web development fundamentals
## Disclaimer

This repository contains assignment prompts and solutions for educational purposes. It is not intended to be run as a standalone application.

## About

This repository showcases my assignments for a Web Technologies course. It demonstrates my progress in front-end development, covering core concepts such as HTML, CSS, JavaScript, and responsive design. Each assignment focuses on applying these concepts in practical exercises.
## Repository Structure

Each assignment is stored in its own folder and contains:
- The assignment prompt
- The solution files
- Any required assets
## Usage

Clone the repository and explore each assignment folder to review the prompts and solutions.

```bash
git clone https://github.com/RiuHz/Tech-Web-Assignments.git
```
